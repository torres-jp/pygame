# Ejercicios Básicos de Pygame 🎮

¡Bienvenido/a a este repositorio! Este espacio está dedicado al aprendizaje y desarrollo de diversos ejercicios prácticos y videojuegos básicos utilizando la biblioteca **Pygame** en Python.

El objetivo principal es aprender los fundamentos del desarrollo de videojuegos 2D, incluyendo el manejo de gráficos, eventos, bucles de juego y físicas sencillas.

## 📁 Estructura del Repositorio

A medida que se vayan completando nuevos retos y ejercicios, se irán organizando en carpetas dedicadas. Actualmente contamos con:

*   **[ta-te-ti](file:///c:/Users/Usuari0x/Desktop/Nueva%20carpeta/py-games/ta-te-ti/)**: El clásico juego de Tres en Raya (Tic-Tac-Toe).

## 🚀 Requisitos Previos

Para ejecutar los proyectos de este repositorio, necesitarás tener instalado Python y la biblioteca Pygame.

1.  **Instalar Python** (versión 3.x recomendada).
2.  **Instalar Pygame** ejecutando en tu terminal:
    ```bash
    pip install pygame
    ```

## 🕹️ Cómo ejecutar un juego

Navega a la carpeta del ejercicio correspondiente y ejecuta el archivo principal:

```bash
cd ta-te-ti
python tateti.py
```
